# Ghost — Desktop AI Assistant

A smart desktop AI assistant built with Node.js, Electron, and OpenAI.

## Phase 1: Terminal AI Assistant

Ghost responds to messages in the terminal using OpenAI's API.

## Setup

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with your `OPENAI_API_KEY`
4. Run `npm start`

## Tech Stack

- Node.js
- OpenAI API
- Electron (coming in Phase 5)
- PostgreSQL + Prisma (coming in Phase 6)
